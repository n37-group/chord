String unification

A string is just a list of characters. An expression is a list of characters or variables. The notation for a variable is just {{X}}. So an expression looks like 'hello {{X}} world'.

The goal of this library is to unify two expressions. For example unifying 'boys and girls' and 'boys {{X}} girls' has an obvious solution, X = 'and'.

The empty string

The empty string, i.e. the empty list is unifiable
